Welcome to First Draft Version of jQuery SQL Designer!

jQuery SQL Designer is web based tool to design, update and retrieve Database Model with drag and drop facility for better user experience. This is beta version of jQuery SQL Designer with limited functionality but in due course I will improve existing features and add more features.

SQL Designer has following features

  • Drag n Drop
  • Table Creation
  • Table Update
  • Column Addition
  • Column Update
  • Declaring Relations
  • Table expand and collapse

SQL Designer makes use of following technologies

  • Seam 2.2
  • jQuery
  • jQuery UI
  • jQuery SVG plug-in
  • Apache DdlUtils
